# Experiment 1

本项目是计算机图形学课程的实验一，通过使用 Taichi Lang，独立推导并实现了 3D 空间中的模型变换（Model）、视图变换（View）和投影变换（Projection）矩阵，成功将三维几何体渲染映射到二维屏幕上。

## 项目简介
本项目模拟了相机观察 3D 物体的完整图形渲染管线前置阶段（MVP 变换）。本实验在基础的 2D 三角形变换之上完成了**选做挑战**，升级为了真正的 3D 透视立方体。用户可以通过键盘实时控制模型旋转。
* **数学核心**: 基于齐次坐标系，推导并实现各个空间变换矩阵，并在 GPU 上并行执行矩阵乘法与透视除法。
* **交互体验**: 键盘监听实时控制模型绕轴旋转，不仅展示了基础的平面旋转，还通过立方体的彩色线框（前红、后绿、侧蓝）完美展现了“近大远小”的真实物理景深与透视效果。

## 项目架构
项目采用标准 `src` 布局，并使用现代化的 `uv` 进行环境管理：
* `src/work1/main.py`: **核心代码与程序入口**。整合了变换矩阵的数学推导函数（`@ti.func`）、GPU 并行顶点计算核心（`@ti.kernel`）以及最终的 GUI 渲染主循环。
* `pyproject.toml` / `.venv`: 项目依赖配置。针对底层图形库兼容性，锁定了 Python 3.11 与特定版本的 Numpy 作为稳定运行环境。

## 代码实现逻辑
1. **数据结构**: 使用 `ti.Vector.field` 在显存中开辟空间，存储几何体的初始顶点位置（坐标归一化在 $[-1,1]$ 之间）以及转换后的二维屏幕坐标。
2. **矩阵构建**:
   - **模型变换** ($M_{model}$): 构建旋转矩阵，赋予模型真正的 3D 旋转自由度。
   - **视图变换** ($M_{view}$): 构建平移矩阵，将相机的世界坐标平移至绝对原点。
   - **投影变换** ($M_{proj}$): 利用视场角（$fov$）和近/远截面距离，先构建透视平截头体挤压矩阵 $M_{persp \to ortho}$，再结合正交投影矩阵 $M_{ortho}$ 组合成完整的透视投影矩阵。
3. **空间映射**:
   - 依据右乘规则组合变换矩阵：$MVP=M_{proj} \times M_{view} \times M_{model}$。
   - 执行**透视除法**（除以齐次坐标的 $w$ 分量），将坐标映射至标准设备坐标
